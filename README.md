# 童趣积分乐园 - 儿童积分任务管理系统

一套面向儿童学习的积分任务管理系统，支持家长端任务管理和儿童端打卡学习。

## 功能特性

### 儿童端
- **知识测试**：首次登录完成测试，根据历史成绩自动调整题目难度（简单/中等/困难）
- **任务打卡**：完成每日任务并上传作业/作品进行打卡
- **积分商城**：使用积分兑换礼品
- **学习进度**：查看每日积分趋势、目标进度、成就徽章
- **个人中心**：管理个人信息、历史测试记录

### 家长端
- **仪表盘**：查看学习数据统计、待审核打卡任务
- **儿童管理**：添加/编辑/删除儿童账号
- **任务管理**：创建/编辑/删除学习任务，支持 AI 生成任务
- **打卡审核**：审核儿童打卡记录，发放积分
- **教材管理**：管理教材和习题册，设置任务页数
- **学习报告**：查看学习数据分析图表

## 技术栈

- **前端框架**：React 18 + Vite 6
- **UI 组件库**：Ant Design 5
- **样式**：TailwindCSS 3
- **状态管理**：Zustand
- **路由管理**：React Router DOM 6
- **数据可视化**：ECharts
- **HTTP 客户端**：Axios

## 登录流程

1. 输入手机号和密码
2. 选择登录角色（家长/儿童）
3. 家长需额外输入二级密码验证
4. 儿童登录后需选择关联的儿童账号
5. 首次登录的儿童需完成知识测试

## 积分体系

### 普通积分
- 每日打卡完成任务可获得基础积分

### 奖励积分
- 按时完成任务：额外 +5~10 分
- 作品质量优秀：额外 +10~20 分
- 完成额外任务：额外 +15~30 分

## 难度自适应

测试系统会根据儿童的历史测试成绩自动调整题目难度：

| 平均成绩 | 难度等级 | 题目类型 |
|----------|----------|----------|
| ≥85分 | 困难 | 复杂运算、地球科学、化学基础 |
| ≥60分 | 中等 | 两位数运算、基础科学常识 |
| <60分 | 简单 | 基础加减法、生活常识 |
